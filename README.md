# Nocturnis

A dark theme for [Obsidian](https://obsidian.md/) with a deep, near-black background and indigo accents.

![screenshot](screenshot.png)

## Features

- Deep dark backgrounds with subtle purple undertones
- Indigo (`#6373ff`) accent color throughout the UI
- Subtle white gradients on headings, bold, and italic text
- Clean blue (`#559de0`) link styling with text color instead of underlines
- Indigo-tinted blockquotes with decorative borders
- Custom checkbox animations
- Dracula-based syntax highlighting for code blocks
- Custom colored tags for `#todo`, `#working-draft`, `#notes`, `#knowledge`, `#writing`, `#ideas`
- Decorative `hr` separators with indigo gradient

## Color Palette

| Role | Color |
|---|---|
| Background (primary) | `#070510` |
| Background (secondary) | `#0d0a14` |
| Text | `#bebebe` |
| Accent | `#6373ff` |
| Links | `#559de0` |
| Headings/Bold/Italic | `#bebebe` to `#ffffff` gradient |

## Installation

### From Community Themes
1. Open Obsidian Settings
2. Go to Appearance > Themes
3. Search for "Nocturnis"
4. Click Install and Enable

### Manual
1. Download `theme.css` and `manifest.json`
2. Create a folder called `Nocturnis` inside your vault's `.obsidian/themes/` directory
3. Place both files in the folder
4. Enable the theme in Settings > Appearance
